# PromptForge

> The Programming Language for AI Prompts.

PromptForge is an open-source compiler and IDE for building, analyzing, optimizing, and deploying AI prompts.

## Features

- Prompt Programming Language (APL)
- Compiler
- Lexer
- Parser
- AST
- Prompt Optimizer
- Prompt Analyzer
- Prompt Score
- Prompt Benchmark
- Multi-LLM Support
- VSCode Extension
- Plugin System

## Example

```apl
prompt "Assistant" {

    role {

        Senior Software Engineer

    }

    task {

        Explain dependency injection.

    }

}
```

Compile

↓

```
You are a Senior Software Engineer.

Task:

Explain dependency injection.
```
